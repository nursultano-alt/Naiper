# AX — Учёт расходов

Удобное PWA-приложение для ежедневного учёта личных расходов. Быстро записывайте траты — приложение само определит категорию.

## Возможности

- ⚡ **Быстрый ввод** — сумма + описание, нажали «Добавить»
- 🤖 **Авто-категории** — приложение определяет категорию по описанию (еда, кофе, транспорт, покупки и др.)
- 📊 **Статистика** — расходы за день, неделю и месяц с разбивкой по категориям
- 📱 **Работает как приложение** — устанавливается на телефон/компьютер через браузер (PWA)
- 🔒 **Оффлайн** — работает без интернета после первой загрузки
- 💾 **Локальное хранение** — все данные хранятся на устройстве в localStorage

## Установка как приложение

### На телефоне (Android/iOS)
1. Откройте сайт в Chrome / Safari
2. Нажмите «Добавить на главный экран» / «Установить»
3. Приложение появится как иконка на рабочем столе

### На компьютере (Chrome/Edge)
1. Откройте сайт
2. Нажмите иконку установки в адресной строке
3. Приложение откроется в отдельном окне

## Запуск для разработки

Откройте `index.html` в браузере или используйте Live Server:

1. Установите [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) в VS Code
2. Откройте проект
3. Правый клик на `index.html` → **Open with Live Server**
