# Naiper — Калькулятор

Современный веб-калькулятор, названный в честь математика Джона Напьера (John Napier).

## Возможности

- Основные арифметические операции: сложение, вычитание, умножение, деление
- Вычисление процентов
- Поддержка клавиатуры
- Адаптивный дизайн для мобильных устройств
- Современный интерфейс с эффектом стекла

## Запуск

Откройте файл `index.html` в браузере или используйте Live Server в VS Code:

1. Установите расширение [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) в VS Code
2. Откройте проект в VS Code
3. Нажмите правой кнопкой мыши на `index.html` → **Open with Live Server**

## Структура проекта

```
Naiper/
```

## Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| `0-9`   | Ввод цифр |
| `.`     | Десятичная точка |
| `+ - * /` | Операторы |
| `Enter` или `=` | Вычислить результат |
| `Backspace` | Удалить последний символ |
| `Escape` | Очистить |
